Angular custom fixed range component

In this case, we receive an array from service with different price values: [1,99, 5,99, 10,99, 30,99, 50,99, 70,99] (or any other values). When the range component receives this values assigns a specific px possition for each which is relative to the range width.
When a bullet reaches a price position, an event is triggered to stop propagation and the price range gets its values set. This Cdk drag event has to be stopped manually, as Cdk doesn't provide it while moving.

Binded price range values to parent:

MIN VALUE: {{priceRange.min | number : '1.2-2'}}€

MAX VALUE: {{priceRange.max | number : '1.2-2'}}€
